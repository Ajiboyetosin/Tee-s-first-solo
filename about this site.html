About this site


This websiteis a tutorial site for the series "How To Build a Website With HTML"on DigitalOcean's community platform. Learn how to recreate and customize this site with HTML by following the step by step instructions in the tutorial series. Participants will finish with an understanding of HTML and a website ready to deploy to the cloud. No coding experience necessary!
